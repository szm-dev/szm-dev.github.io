## Get Started

Getting started with _suzume_ should be a painless process.

### Dependencies

* CMake;
* A C++14 compliant compiler (really, that's all).

### Installing

```
git clone https://github.com/szm-dev/suzume
cd suzume
mkdir build && cd build
cmake ..
make
```
